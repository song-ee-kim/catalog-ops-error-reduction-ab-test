# Review Scraping & Sentiment Analysis

웹에서 상품 리뷰를 수집하고, 감성 분석 및 키워드 분석을 수행한 프로젝트입니다.

## 1. Goal
- 고객 불만 요인 파악
- 상품 개선 포인트 도출

## 2. Workflow
- BeautifulSoup / Selenium 크롤링
- 텍스트 정제
- 감성 분석 (KNU 감성사전 등)
- WordCloud, Keyword Network 시각화
- LDA 토픽 모델링

## 3. Insights
- 부정 리뷰 상위 키워드 분석
- 카테고리별 감성 점수 비교
